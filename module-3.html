Claude Code

It sounds like a developer tool. It kind of is. But it's also one of the most powerful things OO has access to right now — and you don't need to write a single line of code to use it.

What is Claude Code?

Claude Code is a version of Claude that lives inside your file system. Instead of just chatting in a browser window, it can read files, write files, run commands, and build things. Think of it as Claude with hands — it can actually do work, not just describe it.

You access it through the Code tab in Claude Desktop (the downloadable app) or on the Claude web app. It's the same Claude you already know, just with a lot more superpowers turned on. The "code" part refers to what it can do, not what you need to know.

How is it Different from Chat or Cowork?

Claude Chat

Conversation only. Great for ideas, drafts, and questions. Doesn't touch your files or computer.

Claude Cowork

Chat with OO context built in. Best for client work, copy, and agency deliverables.

Claude Code

Can read, write, and run files on your computer. Builds real things: websites, reports, automation scripts.

Getting Started

  • Download Claude Desktop from claude.ai/download — it's free with your Anthropic account.
  • Open Claude Desktop and look for the Code tab in the top navigation.
  • Create a new project folder on your computer for whatever you want to build.
  • In the Code tab, point Claude at your folder and start describing what you want to make.
  • Claude will read existing files, write new ones, and tell you what commands to run when needed.

You can also use Claude Code directly from the Claude web app — look for the same Code tab. The web version has slightly fewer local file permissions but works great for most tasks.

CLAUDE.md — Your Project Briefing File

One of the best features of Claude Code is the CLAUDE.md file. Drop a file with this name into any project folder and Claude Code will read it automatically every time you start a session in that folder. It's how you give Claude standing instructions without repeating yourself.

# CLAUDE.md — OO Project Context

This is a marketing website for Acme HVAC.
Brand voice: friendly, straightforward, never preachy.
Primary color: #E84C1B. Font: Inter.
Always use semantic HTML. No inline styles.
# Ask before deleting any existing files.

Think of CLAUDE.md as the brief you'd give a new contractor on day one. Full Anthropic docs at docs.anthropic.com/claude-code.

OO Use Cases

Use Case 01

AI AE Assistant

Claude Code can pull data from multiple sources, summarize client activity, and draft weekly reports — all from a folder of exports.

Use Case 02

SEO Audit Tool

Point it at a client's site structure and a spreadsheet of keywords, and it can generate a structured SEO gap analysis in minutes.

Use Case 03

Subdomain Deploy

Build a client microsite or landing page in plain HTML/CSS, push it to GitHub, and deploy it to a Cloudflare-hosted subdomain — all in one session.

Use Case 04

Social Content Output

Feed it a content calendar and brand guide and it will generate a full month of social post drafts formatted and ready for review.
